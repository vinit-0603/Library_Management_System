# 📚 Library Management System (Python)

## 🔹 Project Description
This is a simple Library Management System built using Python.  
It allows users to issue and return books while tracking student details and applying fines for late returns.

---

## 🔹 Features
- Manage books using dictionary
- Issue books with student name and issue date
- Return books with fine calculation
- Progressive fine system:
  - Week 1 → ₹10/day/book
  - Week 2 → ₹20/day/book
  - Week 3 → ₹60/day/book
- User-friendly menu system
